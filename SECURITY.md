# Security Policy

## 지원 버전

Phase 0/1 진행 중이며 아직 릴리스 버전이 없음. 모든 보안 리포트는 `main` 브랜치 기준으로 접수.

## 취약점 제보

취약점을 발견하면 **공개 이슈 대신** GitHub Security Advisory 비공개 채널을 통해 제보해주세요:

https://github.com/hansolparkdev/admin-console/security/advisories/new

가능하면 다음을 포함:

- 영향 받는 버전/커밋
- 재현 절차
- 영향 범위 (인증·인가·데이터 유출 여부)
- 가능한 mitigation

## 응답 SLA (best-effort)

- 접수 확인: 영업일 기준 3일
- 심각도 평가: 접수 후 1주
- 패치 + 공개 disclosure: 심각도에 따라 조율

## 범위 밖

- `docs/legacy/` 아래 레거시 문서의 샘플 코드
- 파생 프로젝트(admin-mes / admin-vdi) — 별도 레포에서 각자 정책
